Reset de Senha Esquecida no Windows 11

Aprenda a recuperar o acesso ao seu sistema sem softwares externos, sem pen drives e de forma totalmente nativa. Ideal para situações de emergência.

Atenção Crítica

Este procedimento envolve a manipulação de arquivos do sistema (`System32`). Siga rigorosamente os passos para evitar danos à instalação do Windows. Recomenda-se realizar este processo apenas em computadores de sua propriedade.

1 Iniciando o Modo de Depuração

Para acessar as ferramentas de reparo sem uma senha, precisamos entrar nas Opções Avançadas de Inicialização:

  • • Na tela de login, segure a tecla --> SHIFT.
  • • Com o SHIFT pressionado, clique em --> Reiniciar.
  • • O Windows entrará no ambiente de recuperação. Navegue em:
    Solução de Problemas > Opções Avançadas > Prompt de Comando.

2 Manipulação de Arquivos via Notepad

Como o Prompt de Comando abre em um ambiente limitado (X:), usaremos o Bloco de Notas como um "Explorador de Arquivos" improvisado para renomear arquivos do sistema:

Copiar notepad.exe
  1. No Bloco de Notas, vá em Arquivo > Abrir.
  2. Mude o tipo de arquivo para "Todos os Arquivos (*.*)".
  3. Navegue até C:\Windows\System32 (Nota: a letra da unidade pode variar no modo de recuperação, geralmente é C: ou D:).
  4. Localize o arquivo sethc.exe (Teclas de Aderência) e renomeie-o para sethc.exe.bak.
  5. Localize o arquivo cmd.exe, copie-o e cole na mesma pasta.
  6. Renomeie a cópia do cmd.exe para sethc.exe.

Isso fará com que o Windows abra o Prompt de Comando em vez das Teclas de Aderência na tela de login.

3 Resetando a Senha

Feche tudo e clique em Continuar para o Windows 11. Agora, na tela de login:

  • • Pressione a tecla -SHIFT- cinco vezes rapidamente.
  • • Uma janela preta (Prompt de Comando) abrirá com privilégios de Administrador.

Comando para mudar senha:

Copiar control userpasswords2

Abrirá a janela com o administrador abaixo e o botão para substituir a senha atual.

Comando para identificar usuário:

Copiar net localgroup administrators

Comando para mudar a senha:

Copiar net user [NOME_USUARIO] *

Substitua [NOME_USUARIO] pelo seu nome de conta. Digite a nova senha quando solicitado (ela não aparecerá enquanto você digita).

Conclusão e Reversão

Após entrar no sistema com a nova senha, lembre-se de voltar à pasta System32 e renomear os arquivos de volta ao estado original para manter a segurança e as funcionalidades do sistema intactas.

Passo Final 1 Apague o `sethc.exe` falso (o cmd).
Passo Final 2 Renomeie `sethc.exe.bak` para `sethc.exe`.

Dicas de Segurança Pós-Recuperação

Use Contas Microsoft

Vincular seu login a uma conta Microsoft permite o reset de senha via e-mail ou celular de forma oficial.

Disco de Redefinição

Crie um pendrive de recuperação de senha oficial do Windows enquanto você ainda tem acesso à conta.
